# IMDB Movies Analysis

## Objective
Analyze IMDB movie dataset using SQL to evaluate revenue, popularity, and director performance.

## Tools Used
SQL, MySQL

## Dataset
IMDB movie dataset containing movie details such as title, revenue, rating, genre, and director.

## Steps Performed
- Data exploration using SQL
- JOIN operations across tables
- Aggregation using GROUP BY
- Revenue and rating analysis
- Director performance evaluation

## Key Insights
- Identified top-performing movies by revenue
- Analyzed director performance
- Evaluated movie ratings across categories
